# 🚀 CraftedCV - AI-Powered Resume Builder

CraftedCV is a high-performance full-stack application designed to empower job seekers. By utilizing Generative AI, it helps users craft professional, ATS-optimized resumes that stand out in the modern job market. 🛠️

### ✨ Features

- 🤖 **AI Generation:** Get real-time suggestions for summaries and bullet points.
- 📄 **ATS-Ready:** Layouts specifically engineered to pass through recruitment filters.
- ⚡ **Live Preview:** Watch your resume transform instantly as you edit.
- 📥 **Easy Export:** Generate professional PDF versions in seconds.

### 🛠️ Setup & Installation

1️⃣ **Clone the repo:**

```bash
git clone https://github.com/KedarGhadyalji/craftedcv.git
```

2️⃣ **Install Dependencies:**
Navigate to both Frontend and Backend directories and run:<br> For Frontend:

```bash
cd client
```

For Backend:

```bash
cd server
```

Inside both run:

```bash
npm install
```

3️⃣ **Environment Variables:**
⚙️ Please check the **.env.sample** file in both the frontend and backend folders for the required keys and setup accordingly.

4️⃣ **Launch:**
Run these into your Frontend and Backend Folders to get the app running locally! 🚀
<br> For Frontend:

```bash
npm run dev
```

For Backend:

```bash
npm run server
```
